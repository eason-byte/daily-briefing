01

#1投资思维重塑

🕐 1个月 · 最重要

你不是技术不够，是认知框架有漏洞。先治"心"，再治"术"。

📚 必读书籍（按顺序）
《聪明的投资者》 本杰明·格雷厄姆 → 第8章"市场先生"和第20章"安全边际"反复读三遍
《思考，快与慢》 丹尼尔·卡尼曼 → 理解为什么你追涨杀跌、套住不走
《交易心理学》 马克·道格拉斯 → 为什么明明知道该止损却做不到
🧠 核心认知转变
❌ 错误观念✅ 正确认知
"亏了等它涨回来"亏损是成本，扛单是自杀
"这次不一样"每次都一样，人性不变
"要抓住每个机会"99%的机会可以放弃
"赚钱的交易就是好交易"坏交易也可能赚钱（运气），好交易也可能亏钱（概率）
02

#2基础知识体系

🕐 2-4周

中国语境下的投资框架，比纯西方理论更适合A股实战。

📚 必读书籍
《投资中最简单的事》 邱国鹭 ⭐ 强烈推荐！A股本土化最经典，把投资简化到可执行
《股市进阶之道》 李杰（水晶苍蝇拍） → 中国语境下的投资体系构建
《投资最重要的事》 霍华德·马克斯 → 18个核心原则，每个值得背诵
🎯 重点理解
好公司 vs 好价格 — 九典制药和伊力特，哪个是好公司？估值合理吗？
能力圈 — 不碰不懂的行业，不跟风炒概念
市场周期 — 你在2025-2026大牛市高位冲进去，本身就输了一半
03

#3技术分析基础

🕐 2-3周 · 够用就行

⚠️ 不要沉迷技术分析！它是工具不是信仰。

📚 书籍
《股市趋势技术分析》 罗伯特·爱德华兹 → K线形态、趋势线、支撑阻力
《日本蜡烛图技术》 史蒂夫·尼森 → K线语言入门，一本够了
📊 必学5个指标（多了没用）
📉 MA20/MA60 均线 📈 MACD 趋势反转 📊 成交量确认突破 📍 支撑/阻力位 📏 RSI 超买超卖
💡 对你的意义

你现在的持仓全部是"越跌越买"一路套下来的。学会技术分析后你会知道——下跌趋势中不要加仓，等出现底部结构再动手。 恒生科技ETF从高点跌了多少？如果看均线，早就该止损而不是一路补仓。

05

#5交易系统构建 ⭐

🕐 2-3个月 · 最核心

停止"凭感觉"或"学一招用一招"，建立完整的选股→买入→卖出→复盘闭环。

📚 书籍
《交易系统与方法》 佩里·考夫曼 → 系统化圣经，建立完整流程
《海龟交易法则》 柯蒂斯·费思 → 仓位管理、止损纪律、系统信仰
《澄明之境》 青泽 → 交易哲学和戒律，比任何技术书都重要
⚙️ 你的交易系统模板
环节问题你的规则
选股买什么？用选股漏斗筛选（基本面+趋势结合）
买入什么时候买？等回调、等底部结构、分批建仓
仓位买多少？单只不超过总仓位15-20%
止损亏多少走？硬性止损线 -8%，无条件斩仓
止盈赚多少走？分批止盈、移动止损
复盘怎么改进？每笔交易记录 + 每周复盘
🔥 你现在最缺的就是"止损"

你所有持仓深套，都是因为没有硬性止损线。

07

#7实战训练

🕐 贯穿全程

学得再多，不上场等于零。循序渐进入市。

🏋️ 循序渐进
1
模拟盘 1个月
验证系统胜率
2
小资金 3个月
总资金10%（1-2万）
3
逐步加码
连续6个月正收益
∞
持续复盘
每笔记录+每周复盘
✅ 现在就做的4件事
① 清理现有持仓 — 每只股票问自己：如果现在没持有，我还会买入吗？不会，就卖出。最痛但最有用的方法
② 建立交易日志 — 每笔记录：买入理由、卖出理由、盈亏、哪里对了/错了
③ 每周复盘 — 查漏补缺，不断优化系统
④ 每月读一本书 — 按书单顺序，一本一本啃

🎯 对你的特别总结

你现在是最危险的状态——有5-6年经验，赚过钱也亏过钱，以为自己懂了但其实框架不完整。这个阶段最容易犯的错是"学了一个新指标就觉得自己能翻盘"，然后继续亏。

你只需要做好两件事，就能大幅改善当前情况：

1. 学会止损 — 把 -8% 止损刻进基因

2. 学会等待 — 空仓不是失败，等待才是纪律

按你的风险偏好（可接受本金损失追求高收益），趋势交易+基本面选股的混合风格最适合你。这条路是对的，但你需要在止损和仓位管理上补课。
